# MacDynamicIslandPet 🐾

> 一个可爱的Mac桌面精灵宠物，会在你的屏幕上自由活动，陪你聊天，记住你的重要事件，还会自主思考发表观点！

## 📦 安装

### 从源码构建

1. **克隆项目**
```bash
git clone https://github.com/YOUR_USERNAME/MacDynamicIslandPet.git
cd MacDynamicIslandPet
```

2. **打开Xcode项目**
```bash
open MacDynamicIslandPet.xcodeproj
```

3. **构建并运行**
- 点击 Xcode 左上角的运行按钮
- 或使用快捷键 `Cmd + B` 构建，`Cmd + R` 运行

### ⚙️ 配置（可选）

如需启用AI智能功能（对话、评论、自主思考），需要配置API：

1. **创建配置目录**
```bash
mkdir -p ~/Library/Application\ Support/MacDynamicIslandPet
```

2. **复制配置模板**
```bash
cp MacDynamicIslandPet/config.template.json ~/Library/Application\ Support/MacDynamicIslandPet/config.json
```

3. **编辑配置文件**
```bash
nano ~/Library/Application\ Support/MacDynamicIslandPet/config.json
```

配置文件示例：
```json
{
  "apiKey": "your-api-key-here",
  "model": "claude-sonnet-4-6",
  "personality": {
    "chatEnthusiasm": 60,
    "expressionStyle": 50,
    "focusAttention": 70,
    "thoughtDepth": 55,
    "teasingTendency": 40
  }
}
```

## 🎮 使用指南

### 基础操作

| 操作 | 方法 |
|------|------|
| 显示精灵 | 菜单栏爪子图标 → 显示/隐藏宠物，或 `Cmd+Shift+P` |
| 隐藏精灵 | 同上，或点击精灵后选择隐藏 |
| 拍一拍 | 点击精灵 → 选择"拍一拍" 👆 |
| 聊一聊 | 点击精灵 → 选择"聊一聊" 🗨️ |
| 拖动精灵 | 拖拽精灵到任意位置 |
| 查看等级 | 鼠标悬停在精灵上 |

### 菜单功能

点击菜单栏的 🐾 爪子图标：

- **显示/隐藏宠物** - 快速开关精灵显示
- **添加事件...** - 添加重要日期和事件（精灵会提醒你）
- **事件时间线...** - 查看所有添加的事件
- **精灵兴趣...** - 设置精灵关注的新闻领域
- **设置...** - 配置API和性格参数
- **关于** - 查看项目信息
- **退出** - 关闭应用

### 精灵行为说明

精灵会自主进行以下活动：

| 行为 | 描述 | 触发频率 |
|------|------|----------|
| 🚶 探索 | 在屏幕上走动，发现装饰物 | ~3分钟/次 |
| 😴 睡觉 | 躺下休息，旁边出现床 | 随机触发 |
| 🍜 吃东西 | 享用美食，旁边出现食物碗 | 随机触发 |
| 💭 评论 | 评论你的屏幕活动 | 屏幕边缘触发 |
| 🧠 思考 | 获取新闻并发表观点 | 每小时自动触发 |
